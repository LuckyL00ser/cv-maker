# CV Maker - A Dead Simple Resume Editor

The goal is straightforward: create a CV editor that is as simple and plain as possible. You shouldn't have to struggle with Word-like templates breaking upon the tiniest change, or multi-stage step-by-step applications that want to charge you a million bucks after you've meticulously filled every textbox and selected a color for each field.

It's completely free and simple to use—results are visible instantly. The price for simplicity is a lack of customization, but the default template is designed to be "enough" for a professional-looking result, especially if you aren't applying for a creative position and don't need a CV that stands out visually.

In case you don't like the default template, instead of downloading the already formatted PDF, get the JSON file and input that structured data into a free AI chatbot, asking it to prepare a file formatted in a way you like. It should handle it well, and you'll still have a file for future modifications.

---
### How to run
```
npm install
npm run dev
```
